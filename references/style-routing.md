# Aesthetic Routing

A card's visual register is chosen BY ITS CONTENT — the card type and the topic's culture —
automatically. The user does not name a style. A 古诗词 card routes to ink + calligraphy; a
data card routes to a Vignelli grid; a single modern quote routes to Hara emptiness. Route
first, then the typographic hierarchy (hero → support → meta) runs inside that register.

Two questions decide the route: (1) what card TYPE is it, and (2) does the TOPIC carry a
culture that overrides the default (古诗词/国学/国风 → ink, always).

## Route by content → register

| Card content | Register | Tokens to apply |
| --- | --- | --- |
| 金句 / 语录 (single modern quote) | Hara 留白 / Swiss | near-white field, one ink tone, push 70%+ to empty space, one quiet quotation mark, tiny meta |
| 多行现代金句 / 段落式观点 (not a list) | Swiss left-rail | one strong sans, flush-left, a thin rule under the hero, generous margin — NOT centered (centering breaks past 2 lines), NOT a numbered grid |
| 古诗词 / 国学 / 国风 / 古文 (any type) | 水墨 / 文人画 | rice-paper ground, ink wash, vertical 书法 for the hero line, 留白, a red 印章 as the one accent, quiet 宋体 meta |
| 干货清单 / tips list | Müller-Brockmann / Vignelli grid | a real grid, numbered rail in one accent, flat color as a coding system, even spacing |
| 步骤 / how-to | Vignelli diagrammatic | numbered vertical flow, one accent, clean connectors, one type canon |
| 数据 / 统计 | Vignelli | one oversized numeral as hero, flat color system, tiny source line, lots of calm space |
| 概念 / 定义 | Swiss grid | term as hero, one-line definition as support, one accent rule, generous margin |
| 读书笔记 | 编辑级 (editorial / book page) | refined serif or serif+sans pair, a quiet rule, book + author header, a single highlighted line |
| 治愈 / 情绪金句 | Rothko color field | a glowing soft-edged color field behind the short quote, type set in the calm margin |
| 商业 / 职场干货 | Vignelli / Müller-Brockmann | authoritative grid, restrained palette, one type canon, no decoration |

## Collision rule (culture + multi-line)

When 古典中文 content also needs a grid (诗词清单, 国学数据, 古文步骤), keep the INK GROUND and
ACCENT (rice paper, 宋体/楷体, one 印章-red) but borrow the GRID STRUCTURE from Vignelli —
vertical 书法 only for a single hero line, horizontal 宋体 for the list rows. Never a Western sans
on rice paper, never neon accent. Single 古典 line → full ink/书法 void; multi-line 古典 → ink
palette + grid structure.

## How to route

1. Identify the card type AND whether the topic is 古诗词/国学/国风/古文 (culture overrides type).
2. Pick the register. Culture wins first: any 古典中文 content → the ink register regardless of
   card type. Otherwise pick by card type.
3. Apply the register's tokens to the field, palette, and type voice; the hierarchy runs inside.
4. Single-line heroes (金句/读书笔记) may use a maximal-void register (Hara/Rothko); multi-line
   cards (清单/步骤/概念/数据-with-context) must use a GRID register — never push a list into a
   maximal void.
5. The user may override; the default is the routed register, never a generic text-on-color.

## Compatibility rule

The host (a card) forbids clutter and demands hierarchy. Only route to registers whose
restraint fits — never apply a maximalist token (Yokoo collage, Klimt ornament, Scher
edge-to-edge) to a card; it destroys the negative space and the reading order.

## The ink register (古诗词 / 国学 cards, the most-missed)

For any 古诗词 / 国学 / 国风 / 古文 content, default to: warm rice-paper (宣纸) ground, the hero
line set as vertical brush 书法, ink-black + paper + one seal-red, a red 印章 near the
attribution, generous 留白, and a quiet 宋体/楷体 for source and notes. No gradients, no neon,
no Western grid feel. See the East-Asian section of the master library for the tokens.
